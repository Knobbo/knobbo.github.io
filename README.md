# Big Juju's Mechanic Service Calculator - Modernized

## 🚀 Modern Features Added
- **Glassmorphism & Gradients**: Ultra-modern dark theme with backdrop blur.
- **Smooth Animations**: Fade-ins, hovers, scale effects, shimmer buttons.
- **CSS Variables**: Easy theming.
- **Google Fonts**: Inter font loaded for perfect typography.
- **Enhanced Responsive**: Perfect on mobile/desktop.
- **Debounced Inputs**: Smoother performance.
- **Accessibility**: Reduced motion support, focus states.

## 📱 Live Demo
Open `index.html` in browser or run:
```
npx serve .
```
(Install serve globally if needed: `npm i -g serve`)

## 🎮 Functionality
- Service tab: Items, Upgrades, Repair with live totals.
- Materials tab: Steel/Plastic calculator.
- Grand total + tip percentages (10/15/25/50%).
- Reset button.

Fully functional, deployable to GitHub Pages/Netlify/Vercel.
